## Client.get_address().outputs(address: String, options: OutputOptions)

This [`Client.get_address().outputs(address: String, options: OutputOptions)`](./../api_reference.md#outputsaddress-options-balanceaddressresponse)
expects an address in Bech32 format and will get all outputs that use a given address.  You can use [OutputOptions](./../api_reference.md#outputsoptions) to limit the returned results.
